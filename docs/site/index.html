Command reference

  • aws_explorer — Discover and list AWS resources across accounts and regions.
  • aws_explorer audit — Scan the account for cost waste and security risks (findings linter).
  • aws_explorer bill — Show the account’s bill from Cost Explorer (live –tui).
  • aws_explorer config — Manage the configuration file.
  • aws_explorer config init — Write a starter config.yaml with the built-in defaults.
  • aws_explorer config path — Print the path of the active configuration file.
  • aws_explorer cw — Start the CloudWatch Logs Explorer TUI.
  • aws_explorer ecs — ECS triage helpers.
  • aws_explorer ecs stopped — Triage recently stopped ECS tasks (“why did my task stop?”).
  • aws_explorer expiring — List everything that breaks on a calendar date.
  • aws_explorer find — Fuzzy-find any resource by name, ID, ARN or type.
  • aws_explorer iam — IAM / access debugging helpers.
  • aws_explorer iam can — Simulate IAM policy: “can X do Y on Z?”.
  • aws_explorer iam decode — Decode an “Encoded authorization failure message”.
  • aws_explorer quotas — Service-quota dashboard — limits closest to exhaustion.
  • aws_explorer s3 — Start the S3 Explorer TUI.
  • aws_explorer snapshot-diff — Browse a saved inventory snapshot, or diff two snapshots, offline.
  • aws_explorer summary — List every AWS resource across all regions.
  • aws_explorer trail — CloudTrail activity feed — who did what, and who changed this.
  • aws_explorer vpc — Start the VPC Explorer TUI.
  • aws_explorer whereused — Where-used / blast radius — “can I delete this?”.
